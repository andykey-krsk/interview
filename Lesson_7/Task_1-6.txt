4. Какие типы БД Вы знаете? Приведите примеры.
Реляционные базы данных или SQL базы данных. Хранение данных в этих базах организованно таблицами.
К ним относятся MySQL, MariaDB, Oracle
Не реляционные базы данных или NoSQL базы данных. Хранение данных организовано в виде коллекций.

5. Для чего нужны составные индексы в БД?
Составной индекс применяется для ускорения запросов в базу данных по двум и болле столбцам, если в запросе для фильтрации используется WHERE. Использование индексов ускоряет запросы SELECT и замедляет INSERT  и UPDATE.
